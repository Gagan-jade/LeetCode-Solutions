# Rotate Linked List

## Question
Given the head of a linked list, rotate the list to the right by **k** places.

### Example
**Input:**
```
head = [1,2,3,4,5], k = 2
```

**Output:**
```
[4,5,1,2,3]
```

---

## Explanation
- Start with the list `[1, 2, 3, 4, 5]`.
- Rotate it **2** places to the right:
  - After 1 rotation: `[5, 1, 2, 3, 4]`
  - After 2 rotations: `[4, 5, 1, 2, 3]`.
